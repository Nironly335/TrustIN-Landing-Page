# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Development Setup

No build system — no npm, no bundler, no server required. Open any `.html` file directly in a browser. For mobile testing use Chrome DevTools device emulation or test on real iOS Safari.

## Code Architecture

The project consists of three files (target state):

- `index.html` — structure and semantic markup
- `style.css` — all styles (CSS custom properties in `:root`, mobile-first)
- `script.js` — language switching (`sl()`), form validation (`chk()`, `tog()`), form submit (`sub()`), all copy in the `C` object

**Current state:** everything is embedded in a single file `trustin_v6_form_middle.html`. The task is to split it into separate `index.html` + `style.css` + `script.js`.

### Language system

All UI text lives in the `C` JavaScript object, keyed by locale (`en`, `de`, `ru`, `uk`). `sl(lang, btn)` sets `innerHTML` on every element by ID. When editing copy, update all four locale objects. **Ukrainian (`uk`) is not yet implemented** — only `en`, `de`, `ru` exist.

### Form logic

`chk()` enables the submit button only when: first name non-empty AND last name non-empty AND email contains `@` AND GDPR checkbox is checked. Currently `sub()` logs data to console and shows the success message — no backend.

## 6. Дизайн-система

Общий стиль: чистый, редакционный, доверительный. Не стартап-броский. Целевая аудитория — люди в сложной ситуации, им нужно доверие, не развлечение.

### Цвета (CSS-переменные)

| Переменная | Значение | Использование |
|---|---|---|
| `--a` (`--green`) | `#1A5C3A` | Акцент, кнопки, ссылки, иконки |
| `--al` (`--green-light`) | `#EAF3EE` | Бэкграунд карточек, бейджи |
| `--ab` (`--green-dark`) | `#2E7D52` | Hover-состояние кнопок |
| `--bg` | `#FAFAF7` | Фон страницы |
| `--card` | `#FFFFFF` | Фон карточек |
| `--br` (`--border`) | `#E4E2DC` | Разделители, бордеры |

### Типографика

| Элемент | Шрифт | Размер / вес |
|---|---|---|
| Логотип | Fraunces (serif) | 21px / 700 |
| H1 герой | Fraunces (serif) | clamp(2rem, 4vw, 3rem) / 700 |
| H2 секции | Fraunces (serif) | 1.6rem / 700 |
| H3 карточки | Fraunces (serif) | 1.1rem / 600 |
| Основной текст | Inter | 14–15px / 400 |
| Лейблы, кнопки | Inter | 13px / 500 |
| Микрокопи | Inter | 11–12px / 400 |

Шрифты подключать через Google Fonts с `display=swap`.

### Компоненты

→  Кнопка CTA: высота 48px, border-radius 8px, зелёный фон, белый текст, hover = `--ab`
→  Инпуты: высота 44px, border-radius 7px, border 1px `--br`, focus = border `--a`
→  Карточки шагов: border 1px `--br`, border-radius 12px, padding 24px, фон `--card`
→  Переключатель языков: маленькие кнопки, активный = зелёный фон, неактивный = прозрачный
→  GDPR-чекбокс: кастомный, при checked = зелёный фон с галочкой

### Анимации
Минимум. Только то что не мешает:
→  Fade-in блоков при скролле (IntersectionObserver, opacity 0→1, translateY 20px→0, 0.4s)
→  Hover на кнопке: плавное затемнение 0.15s
→  Появление success-сообщения после отправки формы: fade-in 0.3s
⚠️  Никаких параллакс-эффектов, никаких тяжёлых анимаций. Сайт должен грузиться быстро.
